Il layer di accesso al database attualmente supporta :

Versione 1.0

*** FEATURES ***

- Livello di astrazione sull'accesso al database

- Connessione a singolo database

- Generazione automatica delle query per : SELECT,UPDATE,INSERT,DELETE

- Query SELECT, UPDATE e DELETE con filtri di tipo : EQUALS,NOT_EQUALS,LESS_THAN,LESS_THAN_OR_EQUAL
GREATER_THAN,GREATER_THAN_OR_EQUAL,MATCH_AGAINST,IN,NOT_IN

- Auto riconoscimento delle chiavi delle tabelle e dei campi.

- Query sql diretta

- Unica dipendenza da BasicObject

- Controllo x auto-esecuzione lettura descrizione tabella e database
